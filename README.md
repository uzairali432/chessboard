# chessboard.css

Simple single-page demo showing a chessboard layout using CSS and a small amount of JavaScript.

## Preview

Open [index.html](index.html) in your browser to view the demo.

## Files

- [index.html](index.html) — demo page markup
- [style.css](style.css) — styling for the chessboard and layout
- [script.js](script.js) — minimal JavaScript used by the demo

## Features

- Pure CSS layout for the chessboard grid
- Small, focused JavaScript for interactive behavior or demonstration

## Running locally

No build step required. Open the index.html file directly in your browser, or serve the folder via a simple HTTP server:

```bash
# Python 3
python -m http.server 8000

# Then open http://localhost:8000 in your browser
```

## Development

- Edit styles in [style.css](style.css) to change the board appearance.
- Edit behavior in [script.js](script.js) to modify interactivity.
